Day 02: HTML Forms & Semantic Tags

User Registration Form

HTML 폼 태그와 HTML5 입력 타입을 활용한 사용자 등록 폼 페이지입니다. 시맨틱 태그와 멀티미디어 요소도 포함하여 현대적인 웹 페이지를 구현했습니다.

Registration Form 과제 요구사항

구현 내용

  • 📝 기본 과제
    • 사용자 등록 폼 (이름, 이메일, 비밀번호)
    • HTML5 입력 타입 (date, number, range, email)
    • 라디오 버튼 (성별 선택)
    • 체크박스 (관심사 선택)
    • 드롭다운 메뉴 (국가 선택)
    • 폼 제출/초기화 버튼
  • 🎯 도전 과제
    • 시맨틱 태그: header, nav, section, article, aside, footer
    • SVG 그래픽: 로고 및 아이콘
    • Canvas 요소: 동적 그래픽
    • 멀티미디어: audio, video 태그

학습 목표

  • HTML 폼 요소의 구조와 속성 이해
  • HTML5 새로운 입력 타입 활용
  • 시맨틱 태그로 의미 있는 문서 구조 만들기
  • 멀티미디어 요소 통합
  • 접근성을 고려한 폼 디자인
