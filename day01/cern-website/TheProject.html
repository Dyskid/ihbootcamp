The WWW project

The WorldWideWeb (W3) project allows all links to be made to any information anywhere. This includes other media types and access protocols. The project started in 1989 at CERN as a collaboration system for physicists working around the world.

The WWW world consists of documents, and links. Indexes are special documents which, rather than being read, may be searched. The result of such a search is another ("virtual") document containing links to the documents found.

A program which provides access to the hypertext world we call a browser.
