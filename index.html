🎓 IH Bootcamp - HTML 학습 과제

HTML의 기초부터 고급 기능까지 단계별로 학습하는 프로젝트 모음입니다.

DAY 01

CERN 웹사이트 재현

세계 최초의 웹사이트인 CERN의 초기 웹사이트를 HTML로 재현하는 과제입니다. HTML의 기본 태그와 하이퍼링크를 활용하여 역사적인 웹사이트의 구조를 학습합니다.

CERN 메인 페이지 WWW 프로젝트 Day 01 홈

구현 내용

  • 메인 페이지: The World Wide Web 소개
  • 프로젝트 페이지: WWW 프로젝트 상세 설명
  • 페이지 간 하이퍼링크 연결
  • HTML 기본 태그 활용 (h1, p, a, dl, dt, dd 등)
DAY 02

사용자 등록 폼

HTML 폼 태그와 HTML5 입력 타입을 활용한 사용자 등록 폼 페이지입니다. 시맨틱 태그와 멀티미디어 요소도 포함하여 현대적인 웹 페이지를 구현했습니다.

Registration Form Day 02 홈

구현 내용

  • 📝 기본 폼 요소 (text, email, password, date, number, range)
  • 🔘 라디오 버튼 및 체크박스
  • 📋 드롭다운 메뉴 및 버튼
  • 🏗️ 시맨틱 태그 (header, nav, section, article, aside, footer)
  • 🎨 SVG, Canvas, Audio, Video 요소

Day 02
