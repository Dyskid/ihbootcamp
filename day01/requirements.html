Day 01 - Requirements

과제 개요

오늘 수업에서 HTML의 기본 개념과 역사적 배경을 학습하며, 세계 최초의 웹사이트인 CERN의 초기 웹사이트(https://info.cern.ch/)를 언급했습니다. 이번 과제는 HTML을 활용하여 다음 두 개의 웹페이지를 재현하는 것입니다:

  1. 메인 페이지: https://info.cern.ch/
  2. 프로젝트 페이지: https://info.cern.ch/hypertext/WWW/TheProject.html

이 과제를 통해 HTML의 기본 태그 사용법과 하이퍼링크를 활용한 페이지 연결을 익히고, 실제 역사적인 웹사이트의 구조를 이해합니다.

과제 요구사항

1. 메인 페이지 (index.html)

  • URL: https://info.cern.ch/ 를 참고하여 메인 페이지를 작성하세요.
  • 페이지 제목은 <title> 태그를 사용하여 "The WWW project"로 설정하세요.
  • <h1> 태그를 사용하여 "The World Wide Web"를 제목으로 표시하세요.
  • <p> 태그를 사용하여 간단한 소개 문구를 추가하세요. 예: "WorldWideWeb: Information available from CERN."
  • <a> 태그를 사용하여 "TheProject.html" 페이지로 이동하는 하이퍼링크를 추가하세요.
  • 추가적으로, 원본 페이지의 텍스트와 링크 구조를 최대한 유사하게 재현하세요.

2. 프로젝트 페이지 (TheProject.html)

  • URL: https://info.cern.ch/hypertext/WWW/TheProject.html 를 참고하여 페이지를 작성하세요.
  • 페이지 제목은 <title> 태그를 사용하여 "The WWW project"로 설정하세요.
  • <h1> 태그를 사용하여 "The WWW project"를 제목으로 표시하세요.
  • <p> 태그를 사용하여 WWW 프로젝트의 목적과 특징을 설명하는 문단을 작성하세요. 원본 페이지의 내용을 참고하되, 간결하게 정리해도 괜찮습니다.
  • <a> 태그를 사용하여 메인 페이지(index.html)로 돌아가는 링크를 추가하세요.
  • 원본 페이지에 있는 "What's out there?", "Getting Started" 등의 하이퍼링크를 포함하되, 실제 링크는 동작하지 않아도 됩니다. (예: <a href="#"> 사용 가능)

3. 파일 구조

프로젝트 폴더를 생성하고, 아래와 같은 구조로 파일을 구성하세요:

day01/
├── cern-website/
    ├── index.html
    ├── TheProject.html

두 HTML 파일은 서로 하이퍼링크로 연결되어야 합니다.

도전 과제 (선택)

기본 과제를 완료한 학생들은 아래 도전 과제를 수행하여 img, table, hr 태그를 활용한 추가 구현에 도전하세요.

1. 메인 페이지에 테이블 추가

  • 메인 페이지(index.html)에 <table> 태그를 사용하여 CERN의 주요 프로젝트 목록을 표로 작성하세요.
  • 예: 프로젝트 이름, 설명, 관련 링크를 포함한 3열 테이블.
  • 최소 3개의 문서(html)를 포함하세요 (가상의 프로젝트여도 괜찮습니다).
  • 테이블에는 <thead>와 <tbody>를 사용하고, <th>로 열 제목을 설정하세요.
  • 테이블과 기존 콘텐츠를 구분하기 위해 <hr> 태그를 테이블 위/아래에 추가하세요.

2. 프로젝트 페이지에 추가 이미지와 구분선

  • 프로젝트 페이지(TheProject.html)에 <img> 태그를 사용하여 CERN 관련 이미지를 하나 이상 추가하세요.
  • 각 섹션(예: "What's out there?", "Getting Started") 사이에 <hr> 태그를 사용하여 구분선을 추가하세요.
  • 이미지에는 alt 속성을 반드시 포함하고, 이미지가 페이지 콘텐츠와 조화를 이루도록 배치하세요.

3. 파일 구조

프로젝트 폴더를 생성하고, 아래와 같은 구조로 파일을 구성하세요:

day01/
├── cern-website/
    ├── index.html
    ├── TheProject.html
    ├── images/
        ├── logo.png
