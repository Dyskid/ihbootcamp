🌐 CERN 웹사이트 재현 프로젝트

1991년 팀 버너스리가 만든 세계 최초의 웹사이트를 직접 구현해보면서 HTML의 역사와 기본 구조를 체험합니다. 단순해 보이지만 웹의 모든 기초가 담겨있는 역사적인 프로젝트입니다.

🏠 CERN 메인 페이지

World Wide Web 프로젝트의 시작점입니다. HTML의 기본 구조와 하이퍼링크의 개념을 학습할 수 있습니다.

메인 페이지 보기

주요 학습 내용:

  • HTML 문서 구조
  • 제목 태그 (h1)
  • 문단 태그 (p)
  • 하이퍼링크 (a)
  • 정의 목록 (dl, dt, dd)

📄 WWW 프로젝트 페이지

World Wide Web 프로젝트에 대한 상세한 설명이 담긴 페이지입니다. 더 다양한 HTML 태그들을 활용해 구조화된 문서를 만들어봅니다.

프로젝트 페이지 보기

주요 학습 내용:

  • 소제목 태그 (h2)
  • 구분선 (hr)
  • 주소 태그 (address)
  • 페이지 간 네비게이션
  • 문서 구조화
